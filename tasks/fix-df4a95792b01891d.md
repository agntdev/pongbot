# fix-df4a95792b01891d — Stale `start.json` test files conflict with E2T1's /start implementation

**Weight:** 0.0000 (share of project budget)
**Reward:** 0 PONG

`tests/specs/start.json` expects `/start` to reply `"Welcome! I am ready to help."` and `tests/commands/start.json` declares `["/start"]`. E2T1 (and T02 before it) changed `/start` to reply `"Hi — I'm PongBot! Send /ping to get a pong 🏓 and increment the global counter."` The stale files predate E2 and were not removed when E2T1 was merged. Since the tests-gate globs all `tests/specs/*.json`, the `start.json` spec expects a response the bot no longer produces, causing test failures. These two files should be deleted.
